# Lakecube Excel Plugin

A Databricks Excel Add-in extension that delivers **Smart View parity**:

- Ad-hoc grid: zoom in/out/to-bottom, keep-only, remove-only, pivot row↔column
- Free-form member entry with partial-match + alias resolution
- Member selector dialog (hierarchy tree + attributes + UDAs)
- Persistent POV toolbar + alias-table switcher
- Drill-through to source rows
- Cell comments & linked reporting objects (stored in Lakebase)
- Grid formulas for local what-if (`=$Sales*1.05`)
- Business-rule invocation (right-click → Run Rule)
- Submit selected range; validate before submit
- Formatting and comments preserved on refresh
- Smart-View keyboard shortcuts mirrored

**Status**: lands in P2.
